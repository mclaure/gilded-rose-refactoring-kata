Solving gilded-rose kata for jasmine specs usage

---
## Installing

1. Download the code
2. Install project dependencies:
```
    npm install
```
3. Install Jasmine:
```
    npm install jasmine -g
```
4. Run the following commnad to run the specs
 
```
    jasmine
```
